# 🚀 Running Locally

Clone the repository

```bash
git clone https://github.com/melfmidlabs/AmidGigantePortfolio.git
```

Navigate into the project

```bash
cd AmidGigantePortfolio
```

Restore packages

```bash
dotnet restore
```

Run the application

```bash
dotnet run
```
